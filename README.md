# Blue Decor Studio

This site is configured for deployment to GitHub Pages.

## Deployment

1. Push the repository to GitHub.
2. Enable GitHub Pages in the repository settings.
3. Choose the GitHub Actions deployment source.
4. The workflow in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) will build and publish the site automatically.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
